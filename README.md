# DQN Implementation from Scratch

A complete implementation of Deep Q-Networks (DQN) using only Python and NumPy.

## Quick Start

```bash
# Install dependencies
pip install numpy matplotlib

# Run the example
python main.py

dqn_implementation/

├── utils.py              # Plotting and analysis utilities
├── main.py               # Main execution and examples
└── requirements.txt      # Dependencies


DEPENDENCIES:
- Python 3.6+
- NumPy
- Matplotlib
